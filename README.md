# Jahreskalender Sachsen – Progressive Web App

Studienprojekt zur Entwicklung einer responsiven Progressive Web App zur Darstellung eines Jahreskalenders für das Bundesland Sachsen (2000–2099).

---

## Überblick
Dieses Projekt entstand im Rahmen eines Hochschulmoduls.
Die Entwicklung erfolgte als Einzelprojekt.
Ziel war die Umsetzung einer dynamischen Webanwendung mit Fokus auf JavaScript-Logik, Datenverarbeitung und responsivem UI.

---

## Features
- Darstellung eines Jahreskalenders für die Jahre 2000–2099
- Dynamische Berechnung beweglicher Feiertage mit JavaScript
- Hervorhebung des aktuellen Tages
- Jahresauswahl mit nutzerfreundlichen Voreinstellungen
- Asynchrone Einbindung externer Feiertagsdaten (JSON, AJAX)
- Installierbarkeit und Offline-Nutzung als Progressive Web App
